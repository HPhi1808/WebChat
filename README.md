# 💬 Realtime Chat Application
## 1. Giới thiệu:
Dự án Web Chat Realtime sử dụng Node.js, Socket.io và Supabase. Hỗ trợ đăng ký (xác thực OTP qua Email), đăng nhập phân quyền User/Admin và chat thời gian thực.

### 🚀 Công nghệ sử dụng

- **Backend:** Node.js, Express
- **Database:** Supabase (PostgreSQL)
- **Realtime:** Socket.io (Đang phát triển)
- **Mail Service:** Resend API
- **Frontend:** HTML5, CSS3, Bootstrap 5, Vanilla JS


## 2. Cách chạy dự án:

1. Clone dự án về máy:

            git clone https://github.com/HPhi1808/WebChat.git

2. Đổi file .env.example thành .env và thêm nội dung

3. Cài đặt môi trường:

            npm install
  
4. Chạy dự án:

            npm start
